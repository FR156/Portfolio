Compact Floating Navbar with Tooltips

This navbar stays compact and shows tooltips on hover instead of expanding. Perfect for minimal UI designs.

Features

  • Compact design (5 links only)
  • Shows tooltips on hover instead of expanding
  • Floating position on left side
  • Clean, modern design with subtle animations

How It Works

Tooltip System

Each link has a hidden tooltip that appears on hover with a smooth fade-in effect.

Compact Design

Navbar maintains a fixed width of 4rem (w-16) and centers all icons for a clean look.

Positioning

Uses fixed positioning with transform to center vertically on the page.

// Key CSS for tooltips:

.tooltip {

position: absolute;

left: 100%; /* Position to the right of icon */

opacity: 0;

visibility: hidden;

transition: opacity 0.2s ease;

}

.nav-link:hover .tooltip {

opacity: 1;

visibility: visible;

}

Implementation Instructions

  1. Each navigation link is wrapped in a nav-link relative container
  2. The tooltip is a sibling element to the link, positioned absolutely
  3. Tooltips appear on hover using CSS :hover pseudo-class
  4. All icons are centered with justify-center
  5. The navbar is vertically centered with top-1/2 -translate-y-1/2

Hover over the icons in the floating navbar on the left to see the tooltips appear.
